# Cricket-Chatbot

## An AI chatbot focused on cricket using Retrieval Augmented Generation (RAG).

## Using Llama 2-
Model Used - 'TheBloke/Llama-2-7B-Chat-GGML' [Quantized llama 2 7B chat model]
Can be used to run offline in local CPU.
